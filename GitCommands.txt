1) git init command is used to generate a new, empty Git repository or to reinitialize an existing one
2) git add is a command, which adds changes in the working directory to the staging area. 
3) git commit it records changes to the repository.
4) git reflog  manages the information regarding reference logs.
5) git checkout  is used to switch branches 
6) git branch can be used to create, list and delete branches.
7) git fetch command is used to download commits, files and references from a remote repository into the local repository.
8) git push  updates remote refs along with associated objects.The arguments provided with the command are used to find the location of remote repository
9) git status command shows the state of the working directory and the staging area
10) git clone is used to copy the existing repository into a new repository.
11) git pull command fetches and downloads content from the remote repository and integrates changes into the local repository.
12) git bisect  uses binary search to find the commit that introduced a bug.
